# YTqwery

Полноценное веб-приложение для анализа YouTube, готовое для публикации на GitHub Pages.

## Что реализовано

- Полностью статический frontend на React + Vite
- Поиск и фильтрация видео по просмотрам, дате и релевантности
- Аналитика SEO выбранного видео и подбор ключевых слов
- Панель конкурента с каналом и топ-видео
- Защита доступа через простую клиентскую страницу входа
- Построение сайта в папку `docs` для публикации на GitHub Pages

## Запуск локально

```bash
npm install
npm run dev
```

Откройте в браузере:

`http://localhost:5173`

## Публикация на GitHub Pages

Сборка создаёт статический сайт в папке `docs`.

```bash
npm run build
```

После этого настройте GitHub Pages для репозитория в разделе `Settings > Pages`, выбрав ветку `main` и папку `/docs`.

## Парольный доступ

Вход в приложение реализован на клиенте. Это подходит для скрытия проекта от случайных посетителей, но не является полноценной серверной аутентификацией.

Если нужна настоящая защита, используйте внешний сервис: Cloudflare Access, Vercel/Netlify Identity или отдельный приватный хостинг.

## Структура

- `src/` — frontend-приложение
- `src/components/` — UI-компоненты
- `src/data/mockData.ts` — статические данные для поиска и анализа
- `docs/` — готовый статический сайт для GitHub Pages


Полноценное веб-приложение для анализа YouTube, вдохновлённое функционалом BlackQuery.

## Что реализовано

- Поиск видео и ниш в YouTube
- Фильтрация по просмотрам, дате и сортировка
- Анализ SEO для выбранного видео
- Подбор ключевых слов
- Аналитика конкурента и топ-видео
- Светлая современная тема интерфейса
- Backend на Express с мок-данными и возможностью подключения YouTube Data API по ключу

## Запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение в режиме разработки:

```bash
npm run dev
```

3. Откройте в браузере:

`http://localhost:5173`

## Использование YouTube Data API

Для подключения настоящих данных YouTube создайте переменную окружения `YOUTUBE_API_KEY` и перезапустите сервер.

```bash
export YOUTUBE_API_KEY="ВАШ_API_KEY"
npm run dev
```

- `src/components/` — компоненты интерфейса
- `src/data/mockData.ts` — образцы видео и ключевых слов
